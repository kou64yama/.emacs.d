# ~/.emacs.d

My Emacs configurations.

## Requirements

- [Emacs](https://www.gnu.org/software/emacs/) 25+

## Installation

Checkout this repository at `~/.emacs.d`:

```bash
$ git clone https://github.com/kou64yama/.emacs.d.git ~/.emacs.d
```
